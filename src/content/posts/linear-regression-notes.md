线性回归大概是所有机器学习课程的第一课。它足够简单，却已经包含了「模型—损失—优化」这个贯穿整个领域的基本框架。

## 问题设定

给定数据集 $\mathcal{D} = \{(x_i, y_i)\}_{i=1}^{N}$，其中 $x_i \in \mathbb{R}^d$，$y_i \in \mathbb{R}$。线性回归假设输出是输入的线性函数：

$$
\hat{y} = w^\top x + b
$$

我们希望找到参数 $w, b$，使预测值与真实值尽可能接近。

## 损失函数

最常用的是均方误差（MSE）：

$$
\mathcal{L}(w, b) = \frac{1}{N}\sum_{i=1}^{N}\left(w^\top x_i + b - y_i\right)^2
$$

> 为什么是平方而不是绝对值？一个经典的解释：假设噪声服从高斯分布，最小化 MSE 等价于对参数做极大似然估计。

## 解析解

把偏置吸收进权重（给 $x$ 增加一维常数 1），损失写成矩阵形式 $\mathcal{L} = \|Xw - y\|^2$，对 $w$ 求导并令其为零：

$$
w^* = (X^\top X)^{-1} X^\top y
$$

这就是**正规方程**。当特征维度很高时，求逆的复杂度 $O(d^3)$ 不可接受，此时用梯度下降。

## 梯度下降实现

```python
import numpy as np

def linear_regression(X, y, lr=0.01, epochs=1000):
    n, d = X.shape
    w = np.zeros(d)
    b = 0.0
    for _ in range(epochs):
        y_hat = X @ w + b
        grad_w = 2 / n * X.T @ (y_hat - y)
        grad_b = 2 / n * np.sum(y_hat - y)
        w -= lr * grad_w
        b -= lr * grad_b
    return w, b
```

几个实践中的要点：

1. **特征归一化**：各维尺度差异大时，损失面呈狭长椭圆，梯度下降收敛很慢；
2. **学习率**：太大发散，太小龟速，可以先用 `lr=0.01` 再按 3 倍上下调整；
3. **正则化**：加上 $\lambda\|w\|^2$ 就是岭回归（Ridge），能缓解 $X^\top X$ 不可逆的问题。

## 小结

| 方法 | 复杂度 | 适用场景 |
| --- | --- | --- |
| 正规方程 | $O(nd^2 + d^3)$ | 特征维度低 |
| 梯度下降 | $O(nd)$ / 轮 | 大规模数据 |

线性回归之后，把输出套一个 sigmoid 就得到逻辑回归，把特征做非线性映射就通向核方法与神经网络——这也是后面几篇笔记的主题。
